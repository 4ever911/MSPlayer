## Installation

MSPlayer is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:
swift 3.0 use version 1.x.x <br/>
<br> swift 4.0 use version 2.x.x(not yet) <br/>
```ruby
pod 'MSPlayer', '1.0.4'
```
